== Description ==

Do you want your home page to fade in like google.com does?  I know I do!  This plugin makes it happen for you.  REQUIRES JAVASCRIPT.

== Installation ==

1. Upload `fade_in_like_google` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= What does this plugin do? =

It makes all of the wordpress home pages on your site fade in, just like google.

= Every page? =

No, just the pages that return true for the is_home() function.
